# Philosophy → Mechanism Map

- **Bicameral**: implemented as reservoir (right brain) + Newtonian predictor (left brain).
- **Thermodynamics**: `update_thermodynamics()` modifies forgetting/temperature.
- **Hippocampus**: HDC/VSA memory writes/recall affect learning updates.
- **Identity/telepathy**: synaptic hashes + continuity hash checks.

- **Telepathy beyond hash comparison**: no cross-run/shared state transfer.
- **Philosophical agency**: no explicit decision loop beyond anomaly reporting.
