# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.4.x   | :white_check_mark: |
| < 0.4   | :x:                |

## Reporting a Vulnerability

Please report sensitive issues via email to security@example.com (or your method).
Do not open GitHub issues for security vulnerabilities.

## Secrets handling
- Never commit `.env` files.
- Use `GOOGLE_API_KEY` via environment variables only.
